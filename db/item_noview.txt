// 装备外观隐藏数据库
// 在这里可以设置某些装备, 在特定的时机下不显示他们的外观.
//
// 此数据库的设计背景:
// 龙王蛮蛮所制作的部分图档在穿戴后虽然角色外观看起来正常, 
// 但是只要打开装备面板去查看的话, 可能就会导致客户端直接崩溃.
// 为了避免客户端崩溃, 所以添加此选项.
//
// 此数据库可以使用 @reloaditemdb 重新读取
// 但是若修改的是标记 1 或 2, 那么玩家需要小退之后重新进入游戏才能看到效果.
// 若修改的是标记 4 或 8, 则能够直接生效, 无需小退.
//
// 数据库格式说明:
// 物品编号,标记
//
// 以下为'标记'字段的可选值 (掩码选项):
//   1 - 当自己查看自己装备时, 若此道具为普通装备, 则不在装备面板的“普通装备”选项卡中显示外观
//   2 - 当自己查看自己装备时, 若此道具为时装装备, 则不在装备面板的“时装装备”选项卡中显示外观
//   4 - 当别人查看自己装备时, 若此道具为普通装备, 则不在装备面板的“普通装备”选项卡中显示外观
//   8 - 当别人查看自己装备时, 若此道具为时装装备, 则不在装备面板的“时装装备”选项卡中显示外观
//
// 例如:
// 2220,1	// 穿戴渔夫帽时, 当自己查看自己装备的时候, 不在装备面板的“普通装备”选项卡中显示外观
// 2220,4	// 穿戴渔夫帽时, 当别人查看自己装备的时候, 不在装备面板的“普通装备”选项卡中显示外观
// 2220,5	// 穿戴渔夫帽时, 无论自己看还是别人看, 都不在装备面板的“普通装备”选项卡中显示外观
