import机制的说明
--------------------------------------------------------------

这个机制提供了一种管理配置文件新方式, 让您可以省去每次更新或升级服务端后,
都必须重新配置服务端conf文件的麻烦. 

我要怎样才能使用此机制?
--------------------------------------------------------------

将你想要修改的设置项, 放到对应的import文件中即可.
举个例子, 如果你想修改 /battle/exp.conf 中的一个设置项的值:

	// 基础经验值倍率 (百分比选项)
	base_exp_rate: 700

只需要将这个选项, 完整的复制到 /import/battle_conf.txt 中,
这样就可以避免未来更新服务端带来的配置问题.

很灵活, 不是么?
